Envato VideoGen Batch Extension

Generate dozens or hundreds of short videos automatically on Envato Labs — free Chrome tool

What it does

  • Paste multiple prompts at once (separated by blank lines)
  • Automatically fills each prompt, clicks Generate, waits, and downloads the finished video
  • Optional global start & end frame images (uploaded from your computer)
  • Live numbered preview of all prompts in the sidebar
  • Start / Stop controls — pause or cancel any time
  • Auto-handles navigation between generator and history page

How to install

  1. Open Chrome and go to chrome://extensions/
  2. Turn on Developer mode (toggle in top right)
  3. Click Load unpacked
  4. Select the unzipped folder of the extension
  5. Visit https://labs.envato.com/video-gen
  6. Click the puzzle icon (extensions) in the toolbar → find and open Envato VideoGen Batch
  7. The sidebar appears on the right — paste prompts, choose options, click Start Batch

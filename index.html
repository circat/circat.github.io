Envato VideoGen Batch Extension

Generate dozens or hundreds of short videos automatically on Envato Labs – free Chrome extension.

What it does

  • Paste multiple prompts at once (separated by blank lines)
  • Automatically fills prompt, clicks Generate, waits, downloads from history page
  • Optional global start/end frame images from your disk
  • Live preview of prompts as numbered list
  • Start / Stop controls in sidebar

How to install

  1. Open Chrome and go to chrome://extensions/
  2. Turn on Developer mode (top right toggle)
  3. Click Load unpacked
  4. Select the folder with your extension files
  5. Go to https://labs.envato.com/video-gen
  6. Click the extension icon (puzzle piece) → sidebar opens on the right
  7. Paste prompts, choose options, click Start Batch

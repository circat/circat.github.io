# Envato VideoGen Batch Extension

A free Chrome extension that automates batch video generation on [Envato Labs VideoGen](https://labs.envato.com/video-gen).

Paste dozens or hundreds of prompts → the extension automatically:
- Fills each prompt one by one
- Applies global settings (aspect ratio, sound, speech)
- Optionally sets the same start & end frame images for all videos
- Clicks "Generate"
- Waits for completion
- Goes to the history/results page
- Automatically downloads each finished video (clicks the 3rd button in the newest card)
- Continues with the next prompt

Perfect for creators, marketers, or anyone who wants to scale short AI video production quickly.

## Features

- **Batch input**: Paste multiple prompts separated by blank lines
- **Live preview**: Numbered list of all prompts appears instantly below the textarea
- **Global settings**: Aspect ratio (16:9 / 9:16), Sound, Speech
- **Optional frames**: Upload one start frame and/or end frame from your disk (same for all prompts)
- **Auto-download**: Clicks the download button on the history grid automatically
- **Controls**: Start / Stop batch at any time
- **Navigation recovery**: Automatically redirects back from `/history` to the generator page
- **Sidebar UI**: Clean, dark-mode interface on the right side of the page

## Installation

### Method 1: Load as Unpacked Extension (Recommended for testing)

1. Download the ZIP file:  
   [Download Envato-VideoGen-Batch.zip](https://drive.google.com/file/d/1QjI2k1tVn2oabZ1KJTDQeYLYP9BimTUt/view?usp=sharing)

2. Unzip the file to a folder on your computer (e.g. `Envato-VideoGen-Batch`)

3. Open Chrome and go to:  
   `chrome://extensions/`

4. Enable **Developer mode** (toggle in the top right corner)

5. Click **Load unpacked**

6. Select the unzipped folder

7. The extension should now appear in your list (gray puzzle icon if no custom icon)

8. Go to https://labs.envato.com/video-gen

9. Click the puzzle icon (extensions) in the top toolbar → open **Envato VideoGen Batch**

10. The sidebar opens on the right → ready to use!

### Method 2: Future – Chrome Web Store (not yet available)

This extension is not yet published in the Chrome Web Store.  
If/when it gets published, you can install it with one click like any other extension.
